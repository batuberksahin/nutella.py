nutella.py
================

nutella price ticker on your command line. 

![screenshot](http://i.imgur.com/pRgnj63.png)


### installation

```bash
$ (sudo) pip install git+https://github.com/emre/nutella.py.git
```

### running
```bash
$ nutella
```
